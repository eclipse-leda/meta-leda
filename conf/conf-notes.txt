### Shell environment set up for builds. ###
Welcome to Eclipse Kastro Build Environment.

You can now run 

    Root disk images:       bitbake core-image-minimal
    RAUC Update bundle:     bitbake sdv-rauc-bundle

After the build finished, you can run generated qemu image:

    runqemu ovmf

After the boot completed, provision the device:

    ../scripts/provision-device.sh
